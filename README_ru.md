# Менеджер лорбуков (Lorebook Manager)

Расширение для SillyTavern, которое даёт World Info нормальную панель управления: все лорбуки в одном месте — с папками, скрытием, поиском по всем книгам сразу, массовыми операциями над записями и наглядными привязками.

**Важно:** папки и скрытие никогда не трогают сами файлы лорбуков — всё это хранится в настройках расширения. Удалите расширение — и всё вернётся ровно как было. Книги меняют только явные действия (Удалить/Переименовать/Дублировать/массовые операции с записями), и каждое разрушительное действие сначала спрашивает подтверждение. (Но бэкапы всё равно делайте.)

**Примечание:** расширение активно развивается и не идеально. Баги могут случаться и будут. Если что-то сломалось — посмотрите в консоли браузера ошибки с префиксом `[LBM]` и опишите проблему в issue.

## Возможности

### 1. Панель-менеджер
Один попап со всеми вашими лорбуками. Открывается:
- из wand-меню (Расширения ➜ **Менеджер лорбуков**),
- кнопкой 📚 в ящике World Info,
- slash-командой `/lorebooks`,
- кнопкой в настройках расширения.

Панель — модальный диалог с кнопкой **развёртки** (⤢) на весь экран и обратно; состояние запоминается. На телефонах панель всегда полноэкранная, тулбары — иконками, крупные зоны нажатия.

В строке книги: зелёная **точка активации** (тап включает/выключает книгу глобально), название, **счётчик записей** (включено/всего) и **бейджи привязок** — 🌐 глобально, 💬 чат, 👤 основная у персонажа, 👥 дополнительная, 🎭 персона.

### 2. Скрыть, а не удалить
В меню `⋮` любой книги есть **Скрыть**. Скрытая книга:
- исчезает из селекторов самого ST — глобального списка «Active World(s)», выпадашки «Pick to Edit» в редакторе, пикеров лорбуков в карточке персонажа и пикера чат-лорбука;
- остаётся полностью нетронутой на диске и продолжает работать, если была активна или привязана (в менеджере у неё бейдж ⚠ «активна, но скрыта»);
- живёт в приглушённой секции **«Скрытые»** менеджера (тумблер «Показать скрытые»), откуда возвращается одним тапом.

Если книга где-то используется, диалог скрытия перечислит её привязки и предложит заодно деактивировать её глобально.

### 3. Папки
Сворачиваемые папки с цветной полоской и эмодзи-иконкой, вложенность через выбор родителя (с защитой от циклов). Раскладывать книги можно перетаскиванием на десктопе или через пикер «В папку…» где угодно. Счётчики папок показывают реальное число книг с учётом подпапок. Сортировка: по алфавиту или ручная (стрелки ↑↓ у строк).

### 4. Поиск по ВСЕМ книгам
Встроенный поиск ST ищет только внутри открытой книги. Строка поиска менеджера сканирует **все** книги — ключи, вторичные ключи, заголовки и содержимое; несколько слов работают по логике И. Первый поиск один раз индексирует книги (с прогрессом и отменой), дальше всё мгновенно. Результаты сгруппированы по книгам с подсвеченными фрагментами; клик открывает запись в родном редакторе ST — уже отфильтрованную и подсвеченную.

### 5. Массовые операции с записями
Откройте любую книгу внутри менеджера: все записи со статусом (🟢 включена / 🔵 константа / ⚪ выключена), заголовком и ключами. Одиночные записи переключаются тапом по точке, а кнопка **«Выбрать»** включает режим галочек — выделенные записи можно разом включить, выключить или удалить. Кнопки **«Вкл. все» / «Выкл. все»** обрабатывают книгу целиком. Всё работает напрямую с данными книги (а не с постраничным DOM редактора) и перезагружает редактор ST, если книга в нём открыта.

### 6. Привязки
Диалог `⋮` ➜ **Привязки** показывает и меняет, где книга используется: глобальная активация, привязка к текущему чату, основная или дополнительная книга текущего персонажа. Переименование книги через менеджер сохраняет все эти привязки на новое имя (сам ST делает это лишь частично).

### 7. Хозяйство
- **Интерфейс EN/RU** (по умолчанию русский, режим «Авто» следует за языком ST).
- **Экспорт / импорт** данных расширения (папки, распределение, список скрытых) в JSON.
- Кнопка **«Очистить все данные»** — сбрасывает расширение и возвращает скрытые книги в селекторы ST; сами лорбуки не затрагиваются.

## Установка

В SillyTavern: **Extensions ➜ Install extension**, вставьте адрес репозитория:

```
https://github.com/XaYS-101/Lorebook-Manager-Silly-Tavern
```

Без серверного плагина и правок `config.yaml` — расширение полностью клиентское.

Требуется SillyTavern 1.12.0 или новее (разработано и проверено на 1.17.0).

## Как устроено скрытие (техническая справка)

ST строит селекторы лорбуков из списка `world_names`, где каждый `<option>` несёт индекс книги в качестве значения. Расширение удаляет опции скрытых книг (select2 игнорирует скрытие через CSS), никогда не перенумеровывая и не переставляя остальные, и никогда не удаляет выбранную опцию — поэтому скрытие не может незаметно изменить ваш активный набор. Когда ST перестраивает селекторы, MutationObserver-ы накладывают фильтр заново.
